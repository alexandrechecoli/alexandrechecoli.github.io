INFORMAÇÕES SOBRE ORIENTAÇÃO (TCC)

Fico feliz em orientar alunos no TCC, porém algumas condições devem ser satisfeitas para que possamos trabalhar juntos:

1. TEMAS

Segue uma lista dos temas em que eu posso me envolver em orientação.

Pesquisa operacional/otimização combinatória: modelagem (linear/inteira), análise de sensibilidade, métodos para resolução de modelos, heurísticas, meta-heuristicas, etc.
Mineração de dados/processos: aplicações das tarefas de MD, Classificação/ regressão/ agrupamento associação. Combinação de métodos.
Programação: qualquer tarefa que precise de programação para ser resolvida (deve ter um mínimo de complexidade), eu me interesso. Soluções em VBA, Python, C++...criação de aplicações, interfaces gráficas para usuário.
Análise quantitativa: os tópicos principais são os 3 acima, porém se o trabalho usar métodos quantitativos (previsão de demanda, MRP, etc...), podemos conversar.
O que eu não oriento: básicamente qualquer assunto que não envolva métodos quantitavos eu não irei orientar - ergonomia/gestão/sustentabilidade/5s...

2. FERRAMENTAS

Eu não uso o Microsoft Word. O aluno deve aprender a usar o Latex para escrita de documentos científicos. (Baixar o software MikTex e TexStudio , com isso já dá para começar a escrever textos). Usando o Latex, podemos imbutir imagens em pdf nos documentos, então usamos o Software IPE para criação de imagens de alta qualidade. Para gerenciar as referências que são citadas no texto usamos o JabRef. O primeiro passo então é baixar toda essa suite de programas (todos são gratuitos).

3. TIPO DO TRABALHO

Eu oriento trabalhos no formato de artigos científicos (no caso eu fornece o modelo em latex de algum congresso ou revista para o aluno usar), ou na criação de produtos computacionais junto ao relatório (para alunos a partir de 2023).

4. DATAS

O aluno deve me procurar ANTES de começar o TCC 1, para que possamos ter tempo hábil de discutir o tema a ser desenvolvido e o que será feito durante o trabalho. Não aceito alunos que já estão cursando o TCC2.
O aluno não possui férias entre o TCC1 e o TCC2! Se o trabalho estiver atrasado, as entregas deverão continuar.

Se todos esses requisitos forem aceitos, seguiremos o planejamento mostrado na Figura abaixo.
