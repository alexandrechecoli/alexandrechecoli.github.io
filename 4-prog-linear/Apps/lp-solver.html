Sobre mim

Alexandre Checoli Choueiri (alexandrechecoli@ufpr.br)

Sou professor do curso de Engenharia de Produção da Universidade Federal do Paraná (UFPR), e ministro a disciplina de programação linear. Desenvolvi esse aplicativo baseado no programa GUSEK (desenvolvido pelo professor Leandro Magatão e Luiz Betoni) para visualização de modelos de otimização. A ideia principal é usar em aulas, porém pode ser usado para resolver problemas reais (respeitando a limitação de memória do navegador).

Sobre o projeto

WebHIGHS é um editor e resolvedor de modelos de Programação Linear (PL) e Programação Linear Inteira (PLI), no formato CPLEX LP, que roda inteiramente no navegador — sem precisar instalar nada ou enviar o modelo para um servidor.

Tecnologias e referências

  • Gusek — IDE para GLPK (GNU Linear Programming Kit), com editor de modelos e visualização de resultados. Foi a inspiração para a criação deste aplicativo.
    gusek.sourceforge.net
  • HiGHS — solver de otimização linear, inteira-mista e de programação quadrática, de código aberto, que faz toda a resolução dos modelos.
    highs.dev
  • highs-js — compilação do HiGHS para WebAssembly, o que permite rodar o solver inteiro dentro do navegador.
    github.com/lovasoa/highs-js
  • CodeMirror 5 — editor de texto com numeração de linhas, destaque de sintaxe e temas de cores.
    codemirror.net/5
  • Claude (Anthropic) — assistente de IA muito usado no desenvolvimento deste aplicativo. Na verdade...eu fui somente um gerente para o Cloude.
    claude.ai
