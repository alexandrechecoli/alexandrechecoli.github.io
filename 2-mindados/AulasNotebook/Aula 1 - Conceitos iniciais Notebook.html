Aula inicial, com alguns conceitos sobre o Notebook

1.1 O que é Jupyter notebook?¶

Jupyter notebook uma plataforma para computao em Python iterativa https://jupyter.org/ .Como pode ser percebido, o notebook no usado somente para computaao, mas tambm para escrever textos. Os textos so escritos em uma linguagem markdown. O Jupyter pode ser instalado de diversas formas, no nosso curso iremos utilizar ele juntamente com o interpretador Python baixado, de forma que precisamos instal-lo pelo pip.

1.2 Instalação e utilização¶

Instalação¶

Para instalar o Jupyter basta abir o cmd (Windows) e digitar os seguintes comandos:

>> pip install notebook

Iniciar um notebook¶

Em seguida podemos inicializ-lo pelo terminal (cmd), com os comandos:

>> jupyter notebook
OU (se no funcionar)
>> python -m notebook

O terminal deve estar na pasta em que o notebook ser iniciado (ou aberto). Se no souber manipular o terminal verifique a Seo Apndice

Isso vai abrir uma janela no navegador:

E agora possvel criar um novo notebook ou abrir um j salvo.

OBS: Quem est controlando a aplicao o terminal (cmd), de forma que o mesmo no pode ser fechado, ou a aplicao ser cancelada!

1.2 Sobre os textos¶

Os Notebooks funcionam em clulas, sendo que existem 3 tipos destas (no cabealho acima do arquivo):

  1. Markdown
  2. Python
  3. Raw NB convert
  4. Heading

Em nossas aulas usaremos o Markdown e o Python. Quando o Markdown estiver selecionado, significa que estamos usando uma clula de textos. O Jupyter consegue interpretar diversas linguagens de marcao de textos, por exemplo Latex, html, e a prpria linguagem Markdown genrica (mais usada em Notebooks Jupyter). Um guia rpido (mesmo) de tpicos sobre o Markdown pode ser encontrado em https://github.com/adam-p/markdown-here/wiki/Markdown-Cheatsheet

1.3 Sobre os Códigos¶

Ao se alterar o tipo de clula para Python, todo o contedo inserido na clula deve ser de cdigo python, por exemplo:

1.3 Vantagens e desvantagens do Notebook¶

O Notebook uma tima ferramenta para realizar anlise de dados, pois as bases importadas ficam salvas na memria, o que facilita a explorao. As ferramentas textuais tambm auxiliam na construo de uma lgica e mesmo uma "histria" de como a anlise foi sendo desenvolvida. Relatrios ficam muito mais iterativos e didticos, fazendo com que ele seja muito usado na indstria para a prottipao de anlises.

No entanto, quando se deseja implementar algoritmos ou aplicaes mais complexas, os notebooks no so uma boa pedida. Por exemplo, podemos implementar interfaces grficas para o usurio com o Python, usando o pacote Tkinter por exemplo. Nestes casos, a melhor soluo utilizar uma IDE (ou mesmo um editor de cdigo, como o Visual Studio Code)

Apêndice: usando o terminal & Google Colab¶

O terminal (cwd) um aplicativo do Windows que permite acessar informaes, abrir arquivos e programas. Precisamos saber alguns comandos bsicos para manipular o terminal.

Abrir¶

Para abrir o terminal basta selecionar a lupa para busca de programas e digitar cmd. O nome do programa aparece como "Prompt de Comando"

Mudar de diretório¶

O terminal pode "entrar e sair" de pastas, como fazemos com a interface grfica, porm por meio de comandos. A primeira linha do terminal sempre nos diz em qual diretrio estamos no momento, por exemplo:

Para listarmos tudo que tem em um diretrio usamos o comando dir:

Para acessar um diretrio especifico usamos o comando cd seguido do nome do diretrio que desejamos ir, por exemplo:
>> cd "C:\Users\Usuario\Documents\Alexandre"

OBS: Se estamos em uma partio (C: por exemplo) e queremos mudar para outra partio, basta digitar o nome da partio. Por exemplo, se estamos em C: e queremos ir para G: o comando fica:
>> G:

Google Colab¶

Uma opo ao Jupyter Notebook o Google Colab. O Google Colab um aplicativo (web) que executa notebooks, ou seja, com ele no necessrio realizar nenhuma instalao na prpria mquina (nem mesmo Python!). A desvantagem que ele tm um limite de processamento que provavelmente muito mais baixo do que o de computadores pessoais, ainda, necessrio estar conectado internet para utiliz-lo.
