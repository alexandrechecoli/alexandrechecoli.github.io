4.1 NumPy

O NumPy (Numerical Python) um dos pacotes bsicos mais importantes para o processamento numrico em Python. Isso pela sua estrutura de dados principal, o numpy array (array), que faz o processamento de dados de forma muito mais eficiente do que listas, por exemplo.

4.1.1Array NumPy

O array numpy uma estrutura para armazenar dados numricos (em sua maioria), e tem seu funcionamento como um vetor ou lista. Existem diversas formas de se criar o array. Abaixo criamos array de 3 formas distintas: usando uma lista com valores, a partir da funo range() e a funo np.arange() (equivalente ao range do NumPy).

In[1]:
import numpy as np

lista = [1,2,3,4,5] # lista normal

# Array de lista
array1 = np.array(lista)

# Array de range
array2 = np.array(range(10))

# Array de arange
array3 = np.arange(10)

Os arrays em NumPy podem ser processados de forma vetorizada, o que aumenta a eficincia dos clculos. Isso quer dizer que podemos realizar operaes matemticas em todos os elementos da array sem usar laos for (sempre vai existir um lao, porm ele realizado em funes pr-compiladas em C/C++ ou Fortran, embutidas no pacote NumPy). Considere um vetor de 10000 elementos representado por uma lista e por um array, que deve ter seus elementos individuais multiplicados por 2. O cdigo abaixo faz esses clulos e coleta o tempo de processamento de cada um, usando uma lista e um array (com a funo so Notebook %time).

In[2]:
l1 = list(range(100000))
l2 = np.arange(100000)

%time for i in range(len(l1)): l1[i] = l1[i]*2
    
%time l2 = l2 * 2

4.1.6 Indexao booleana

Tambm podemos realizar operaes booleanas em arrays, de forma que o resultado ser um novo array de valores booleanos, de acordo com a condio. Considere o exemplo:

In[19]:
arr_string = np.array(["Dwight", "Michael", "Angela", "Oscar", "Michael", "Angela"])

# Condio : quais elementos do array so iguais a "Michael"?
arr_bool = arr_string == "Michael"
print(arr_bool)

# Condio : quais elementos do array so iguais a "Michael" OU "Angela"
arr_bool = (arr_string == "Michael") | (arr_string == "Angela")
print(arr_bool)

[False  True False False  True False]
[False  True  True False  True  True]

Tambm podemos fazer o processo reverso: passamos um array de booleanos para um array, e ele retorna somente os elementos (ou arrays) em que a condio verdadeira:

In[20]:
arr_string = np.array(["Dwight", "Michael", "Angela", "Oscar", "Michael", "Angela"])
arr_booleano = np.array([True,False,False,True,False,False])

# Seleciona somente os elementos em que arr_booleano == True
print(arr_string[arr_booleano])

['Dwight' 'Oscar']

Tambm podemos fazer a indexao booleana em arrays multidimensionais. Nesses casos as condies verdadeiras retornam arrays de dimenses menores. Considere o seguinte caso:
