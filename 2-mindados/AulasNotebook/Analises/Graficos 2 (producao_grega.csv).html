Gráficos 2 (producao_grega.csv)¶

Considerando o conjunto de dados producao_grega.csv. Este conjunto contm os dados de medio do diametro de peas de cermica peca1 e peca2, da coluna pea, durante uma semana. Os dias so representados numricamente na coluna dia e as medies em Medicao. A coluna Colaborador contm o dado de quem coletou a medida, e Temperatura a temperatura durante a coleta. Os defeitos que foram encontrados (se foram) so anotados na coluna Nome defeito, sendo que a coluna Defeito? atribui 1 para a ocorrncia de defeito na pea coletada. Faa o que se pede (em todos os grficos, insira bordas, altere a cor e coloque legendas nos eixos e ttulos).

Perguntas¶

  1. Quantas amostras foram coletadas por cada Colaborador?

  3. Plote uma figura com dois grficos de barras com a produo de cada colaborador, porm separando pelas duas peas.
  4. Plote a proporo de coletas de pea1 e pea 2 em um grfico de pizza. Destaque a parte do grfico referente ao maior nmero de peas (de forma automtica, sendo que se os dados forem alterados o grfico tambm o ser).
  5. Existe um bom palpite para a distribuio de probabilidade das peas? Plote uma figura com dois histogramas, um para cada pea, para ajudar a visualizar.
  6. Como a distribuio das medidas (por pea) afetada pelos dias da semana? Plote uma figura com dois grficos, um para cada pea, e em cada grfico um histograma para cada dia da semana, referente s medies coletadas.
  7. Existe alguma relao entre as temperaturas e as medies aferidas? Plote uma figura com dois grficos do tipo scatterplot para responder questo.
  8. Sabemos que pelo principio de Pareto 80% de consequncias so devidos 20% de causas. Para visualizar esse principio em relao frequncia de defeitos, crie um grfico de Pareto e verifique quais defeitos deveriam ser tratados primeiro. ### Faa uma analise de Paretto em relao aos defeitos ###
# use ax.twinx para dois eixos:  # https://matplotlib.org/stable/api/_as_gen/matplotlib.axes.Axes.twinx.html

# Criando os dados para o Pareto:
dtPareto = dt[dt["Defeito?"] == 1].groupby("Nome defeito").count()
dtPareto = dtPareto.sort_values(by="Defeito?", ascending = False)
arr_acumulado = dtPareto["Defeito?"].cumsum() 
arr_acumulado = arr_acumulado / dtPareto["Defeito?"].sum()
dtPareto["Acumulado"] = arr_acumulado
#print(dtPareto)

fig, ax = plt.subplots(1,1)
ax.bar(dtPareto.index, dtPareto["Defeito?"])

ax2 = ax.twinx()
ax2.plot(dtPareto["Acumulado"], color = "black", marker = "o")
plt.show()
