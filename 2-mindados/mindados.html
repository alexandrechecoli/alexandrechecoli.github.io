Introdução à mineração de dados

SOBRE

Mineração de dados pode ser considerado o conjunto de técnicas que realizam a extração não trivial de informações implicitas, previamente desconhecidas e potencialmente úteis, a partir de dados. Abaixo estão os links para códigos Notebooks desenvolvidos em aula, exercicios, artigos e conjuntos de dados.

Todos os dados dos notebooks estão presentes no arquivo compactado de dados (link logo abaixo). Além disso, os conjuntos podem ser importados diretamente do site usando o caminho https://alexandrechecoli.github.io/2-mindados/Dataset/NomeDoConjunto/nome_dos_dados.
